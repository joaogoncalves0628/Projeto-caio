# Projeto Caio

Este projeto agora possui um backend local simples para salvar imagens em um banco SQLite e exibi-las na galeria.

## Como rodar

1. Instale as dependências:
   npm install
2. Inicie o servidor:
   npm start
3. Abra o navegador em:
   http://localhost:3000

## Funcionalidades

- Upload de imagens pela interface
- Armazenamento local das imagens na pasta uploads
- Persistência dos registros em SQLite
- Exibição automática das imagens na galeria principal
